java

1.1 版

java command

java [クラス名]
または
java -jar [jar ファイル名]

javac command

javac [java ファイル名]
source
ソースのバージョン 1.4 1.5 5 など
target
指定されたバージョンの VM 上で動作するクラスファイルを生成します。
classpath
クラスパスを指定します。指定しない場合は、「-classpath . 」が指定したものとみなされされます。また、クラスパスを複数指定するには、「;」で区切ります。「.」も指定しない場合は、検索されません。
ea
"-ea" は、システムクラスを除いた、全てのクラスでアサーションを有効にします。

javap command

javap -verbose [クラス名]
java のクラスのバージョンを表示する。

クラスローダー

J2SE で定義されているクラスローダが次の3つです。
  • ブートストラップクラスローダ（初期クラスローダ）：コアパッケージなど (rt.jar, i18n.jar) のクラスローダ
  • 拡張クラスローダ：拡張機能機能のインストール型オプションパッケージ (lib/ext) のクラスローダ
  • ユーザ定義クラスローダ（システムクラスローダ、アプリケーションクラスローダ）
